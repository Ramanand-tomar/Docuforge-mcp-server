# Sample Prompts for AI Document Generation

## Academic Paper

```
Create a document titled "Machine Learning in Healthcare" in markdown format.

Generate an Introduction section:
"Write a scholarly introduction for a research paper on machine learning applications in healthcare. Include context about the current state of healthcare AI, the problem statement regarding diagnostic accuracy, and research objectives."

Generate a Literature Review section:
"Write a literature review covering recent advances in deep learning for medical imaging, NLP for clinical notes, and predictive models for patient outcomes. Cite key papers and trends from 2020-2024."

Generate a Methodology section:
"Describe a methodology for evaluating ML models in a clinical setting, including dataset selection, model architectures (CNN, Transformer), evaluation metrics (AUC-ROC, F1), and cross-validation strategy."

Format the document as "academic" style, then export to PDF.
```

## Professional Resume

```
Create a document titled "John Smith - Senior Software Engineer" in markdown format.

Generate a Professional Summary section:
"Write a professional summary for a senior software engineer with 8 years of experience in full-stack development, cloud architecture (AWS), and team leadership. Highlight expertise in TypeScript, React, Node.js, and microservices."

Generate a Technical Skills section:
"Create a comprehensive technical skills section organized by categories: Languages, Frameworks, Cloud & DevOps, Databases, and Tools."

Generate a Work Experience section:
"Write work experience entries for: 1) Senior Engineer at TechCorp (2021-present) leading a team of 6, 2) Software Engineer at StartupXYZ (2018-2021) building a SaaS platform, 3) Junior Developer at WebAgency (2016-2018)."

Format as "resume" style, then export to PDF.
```

## Business Report

```
Create a document titled "Q4 2024 Performance Report" in markdown format.

Generate an Executive Summary:
"Write an executive summary for a Q4 business report. Revenue grew 23% YoY to $4.2M. Key wins: launched 2 new products, expanded to 3 new markets, reduced churn by 15%. Challenges: supply chain delays, hiring in engineering."

Generate a Financial Overview section:
"Create a financial overview with revenue breakdown by product line, gross margin analysis, operating expenses summary, and comparison to Q3 and Q4 last year."

Generate a Strategic Initiatives section:
"Outline strategic initiatives for Q1 2025 including product roadmap priorities, market expansion plans, and operational efficiency goals."

Format as "report" style, then export to PDF.
```

## Blog Post

```
Create a document titled "The Future of AI-Powered Developer Tools" in markdown format.

Generate an Introduction:
"Write an engaging blog post introduction about how AI is transforming software development. Use a conversational tone, include a hook about productivity gains, and preview what the article will cover."

Generate a section called "The Current Landscape":
"Describe the current state of AI coding assistants, code generation tools, and automated testing. Mention specific examples and their impact on developer workflows."

Generate a section called "What's Next":
"Predict future developments in AI-powered development tools over the next 3-5 years. Cover topics like autonomous coding agents, AI-driven architecture decisions, and natural language programming."

Format as "blog" style, then export to PDF.
```

## Cover Letter

```
Create a document titled "Application - Product Manager at AI Startup" in plain format.

Generate the full cover letter:
"Write a compelling cover letter for a product manager role at a fast-growing AI startup. The candidate has 5 years of PM experience, previously led B2B SaaS products, and has a technical background in CS. Show passion for AI, highlight relevant achievements, and demonstrate culture fit for a startup environment."

Export to PDF.
```
